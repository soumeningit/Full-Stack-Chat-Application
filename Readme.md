# Talk-Time

## Description

Talk-Time is a full stack chat application. Uses Socket.io for real time communication, user can also send file, store the user data in encrypted format in Mongo DB Database.

## Technologies Used

Client - React.js
Server - Node.js,Express.js
Database - MongoDB
